#       二手交易平台 - 搜索模块用例描述





| 用例名称 | 搜索                                                     					|
| -------- | --------------------------------------------------------------------------	|
| 参与者   | 用户                                                         				|
| 触发条件 | 用户输入搜索关键字并点击搜索按钮                         							|
| 前置条件 | 搜索框中内容不为空                                      						|
| 后置条件 | 返回符合用户输入关键字的模糊查询结果的商品列表                                      |
| 正常流程 | 1.用户输入搜索关键字 2.用户点击搜索按钮 3.小程序跳转到包含所有结果的商品列表页 			|
| 扩展流程 |	 若用户无输入或输入空格，并点击搜索按钮，则提示不能输入空白							|
| 特殊需求 | 	系统可以展示搜索历史，和根据搜索历史生成的推荐搜索                                  ｜                         				|

| 触发条件 | 用户想要查看商品 |
| 前置条件 | 用户经过了账户认证 |
| 后置条件 | 
 |

| 扩展流程 | 2a.用户选择商品排序方式。

我的收藏

| 我的收藏 |  |
| --- | --- |
| 参与者 | 用户 |
| 触发条件 | 用户想要收藏商品 |
| 前置条件 | 用户经过了账户认证，要收藏的商品未下架 |
| 后置条件 | 
 |
| 正常流程 | 1.用户请求收藏商品。
2.系统将商品加入“我的收藏”列表。 |
| 扩展流程 |  |
| 特殊需求 | 若我的收藏列表当中的商品下架或者被标志为已交易，则从列表中删除，并向我的消息发送通知。 |
